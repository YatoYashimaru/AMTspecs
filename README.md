# AMTspecs Project by YatoYashimaru <br>

### Required to Install
- chafa
- ascii-image-converter
- cat

### Install

```bash
 git clone https://github.com/YatoYashimaru/AMTspecs
 sudo sh install.sh 
```

### help
```bash
 AMTsp help
```

### Before Use
- run 
```bash
nano ~/.bashrc
```
- add line to give premission
```bash
export PATH="$HOME/.local/bin/AMTspecs:$PATH"
```
